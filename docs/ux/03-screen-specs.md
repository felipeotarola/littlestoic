# UX Artefacts - Screen Specs

## Global Layout

- Top: gentle header bar with progress dots (5 steps)
- Middle: illustration zone (dominant)
- Bottom: choice cards (large tap targets)
- Optional: small helper copy below header (single line)

## Step 1: Theme

- Illustration: abstract mood scene (color + shape)
- Choices: 4 cards, 2x2 grid
- Microcopy: single question line

## Step 2: Character

- Illustration: hero silhouette + subtle motion
- Choices: 4 cards, 2x2 grid
- Show chosen theme as a small badge (non-clickable)

## Step 3: Place

- Illustration: place panorama (parallax layers)
- Choices: 3-4 cards, 2 rows

## Step 4: Event

- Illustration: scene setup with a soft "spark" accent
- Choices: 4 cards, 2x2 grid

## Step 5: Story Experience

- Illustration: full-bleed scene with floating elements
- Text: story blocks (3-5 paragraphs)
- Controls: Play/Pause voice (future), "Next" for gentle scroll
- End: short positive closure card + "Create another"

## Parent Settings Home

- List of settings rows with clear value preview
- Each row navigates to a simple selection screen

## Parent Setting Detail

- Single choice list or toggles
- Save on select
