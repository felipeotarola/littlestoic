# UX Artefacts - Component Inventory

## Core Components

- ProgressDots (5 states)
- IllustrationStage (supports parallax + floating elements)
- ChoiceCard (icon + label + large hit area)
- ChoiceGrid (2x2 or 2xN)
- PrimaryButton (single action)
- SecondaryTextLink (rare)

## Story Components

- StoryParagraph
- StoryTitle (optional)
- EndingCard (reinforces chosen theme)

## Parent Components

- SettingsList
- SettingsRow
- SelectorList
- ToggleChip

## States

- Default / Selected / Disabled
- Loading (skeleton shimmer)
- Error (single-line prompt + retry)
