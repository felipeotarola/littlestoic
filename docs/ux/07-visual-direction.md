# UX Artefacts - Visual Direction

## Typography

- Display: "Fraunces" (soft, storybook serif)
- UI: "Bricolage Grotesque" (friendly, rounded)
- Numerals: tabular off (organic feel)

## Color Palette (Base)

- Cream: #FFF5E6
- Honey: #F3C46C
- Coral: #F27B6A
- Sky: #7CC7E8
- Sage: #8BC8A5
- Cocoa: #5A3E2B (text)

## Illustration Style

- Soft gradients, visible brush texture
- Round silhouettes, no sharp corners
- Characters have oversized heads and simple eyes

## Iconography

- Filled icons with 2px stroke
- Rounded terminals

## Layout Scale

- Corner radius: 20-28
- Spacing unit: 8
- Hit target: 56x56 minimum
